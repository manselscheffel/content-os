# YouTube Description Template

Use this template for every video description. Fill in the dynamic sections, keep the footer consistent.

Footer content (connect links, about text, hashtags) is loaded from config — see `.claude/content-os.local.md`.

---

## Template

```
[2-3 sentence summary of what this video covers. Lead with the value — what will the viewer learn or gain? Don't start with "In this video..." — start with the insight.]

=== TIMESTAMPS ===
[Generated by extract_timestamps.py — add after publishing]
0:00 - Intro
[...]

=== RESOURCES MENTIONED ===
[List tools, repos, links referenced in the video. Use bullet format.]

=== CONNECT ===
[Loaded from config — your community, social, and newsletter links]

=== ABOUT ===
[Loaded from config — your about text]

[Hashtags — loaded from config, max 5 total]
```

## Rules

- Summary: 2-3 sentences max. Lead with insight, not "In this video..."
- Always make it SEO driven for the first 200 chars.
- Timestamps: Always include if video is 5+ minutes
- Resources: Only list things actually mentioned. Include direct links where possible.
- Footer: Keep consistent across ALL videos. Connect and About sections are loaded from config — never hardcode them in scripts.
- Hashtags: Max 5. Rotate based on topic. Base set is configured in `.claude/content-os.local.md`.
- No affiliate disclaimers unless there are actual affiliate links.
